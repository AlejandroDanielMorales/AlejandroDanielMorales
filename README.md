## 📚 **Habilidades en React**  

- **Hooks**: Uso intensivo de `useState`, `useEffect`, `useContext`, y `useReducer` para gestionar el estado de la aplicación de forma eficiente.
- **Peticiones Asincrónicas**: Integración de APIs externas utilizando `axios` y `fetch`, junto con el manejo de estados de carga y errores.
- **Enrutamiento con React Router**: Implementación de rutas dinámicas, protegidas y anidadas para crear aplicaciones SPA (Single Page Application) de manera fluida.
- **Gestión de Estados Globales**: Utilización de **Context API** para manejar estados globales y **Redux** para aplicaciones más complejas.
- **Manejo de Formularios**: Uso de **Formik** y **Yup** para la validación y manejo de formularios complejos y dinámicos.

## 🧑‍💻 **Proyectos Destacados**  
### 🌍 **Proyecto 1: E-commerce React App**
- **Descripción**: Desarrollé una tienda online con React.js utilizando **React Router**, **Axios** para las peticiones API, y **Formik** para la gestión de formularios de compra.
- **Características**:
  - Navegación fluida entre páginas con React Router.
  - Implementación de filtros y búsqueda en tiempo real.
  - Integración de pasarela de pago simulada.
  
### 📱 **Proyecto 2: App de Tareas**
- **Descripción**: Una app de tareas desarrollada con React.js que permite a los usuarios agregar, editar y eliminar tareas. Utiliza **Redux** para la gestión de estado global.
- **Características**:
  - Sincronización del estado entre diferentes componentes con Redux.
  - Implementación de un sistema de autenticación con JWT.
  - Almacenamiento en el navegador con **localStorage**.
